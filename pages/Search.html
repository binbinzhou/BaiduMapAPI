检索 BaiduMap API

检索功能

在指定区域内检索，例如圆形、矩形、多边形...，

例如：在地图绘制圆形覆盖物：

	//绘制圆形
	var opts = {
		fillColor:"blue",
		strokeWeight: 1,
		fillOpacity: 0.3,
		strokeOpacity: 0.3
	};
	var circle = new BMap.Circle(point,1500,opts);
	map.addOverlay(circle);

创建本地检索对象，在指定范围内进行检索

	//搜索公园关键字
	var optss = {
		renderOptions:{
			map:map,
			autoViewport:false
		}
	};
	var local = new BMap.LocalSearch(map,optss);
	local.searchNearby('公园',point,1500);

关键字检索

	var local = new BMap.LocalSearch(map,opts);
	local.search('公园');

多关键字检索

	var local = new BMap.LocalSearch(map,opts);
	local.searchInBounds(['酒店','公园'], map.getBounds());

输入框搜索，相当于将输入的值传递进单个关键字搜索的方法中，

	local.search(input.value);
