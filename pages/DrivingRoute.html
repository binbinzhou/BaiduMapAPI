驾车路线 BaiduMap API

驾车路线

根据起点终点名称查询驾车路线：

	//设置驾车路线，可设置显示检索结果面板
	var opts = {
		renderOptions:{
			map:map,
			autoViewport:true,
			panel:'r-result'
		}
	};
	//创建驾车路线对象
	var driving = new BMap.DrivingRoute(map,opts);
	
	//调用检索方法查询到驾车路线
	driving.search('中山公园','大世界');

根据起点终点坐标查询驾车路线：

	//创建驾车路线对象
	var driving = new BMap.DrivingRoute(map,opts);
	
	//调用检索方法查询到驾车路线
	driving.search(pointA,pointB);
